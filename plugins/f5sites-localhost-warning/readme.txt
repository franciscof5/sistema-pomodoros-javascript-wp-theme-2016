[F5 Sites | Localhost Warning](

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'F5 Sites | Localhost Warning'
3. Activate it from your Plugins page.
4. Active and configure hostnames on options for customize warn css for each hostname, up to 3 hostnames.
