== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'F5 Sites | Footer Bar'
3. Activate it from your Plugins page.
4. Configure ir direct on plugin folder, edit f5sites-footer-bar.php
