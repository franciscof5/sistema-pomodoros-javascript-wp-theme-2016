== Description ==

Ranking Calendar is a of the visualization plugin for your WordPress site which creates yearly calendar for your posts.

Create a new page (having 'no sidebars' template) for your calendar ranking and insert the shortcode [ranking-calendar] in the editor. Load this page and enjoy the view.

Original plugin: http://www.sanisoft.com/blog/2009/08/21/wordpress-plugin-calendar-archives/

== Installation ==

1. Upload directory 'ranking-calendar' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin options by clicking on 'Settings' link near plugin name or 'Settings' -> 'Ranking Calendar' navigation link. Make sure that your plugin directory must be writable by webserver to use caching of pages and images
4. Create page having [ranking-calendar] shortcode. It will be better to use empty (having no sidebars) template for this page
